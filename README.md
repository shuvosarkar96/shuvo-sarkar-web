# Shuvo's Portfolio

Personal portfolio website showcasing my projects and experience.

**Live:** [shuvosarkar.vercel.app](https://shuvosarkar.vercel.app/)

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Vercel (Deployment)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Project Structure

```
app/
├── page.tsx          # Main landing page
├── layout.tsx        # Root layout
└── ...
public/               # Static assets
```

## Deployment

Automatically deployed to Vercel on push to `main`. 

To deploy manually:
```bash
vercel --prod
```
